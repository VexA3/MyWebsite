Disclaimer:
This tracker may not work as advertised within about 1 hour and 30 minutes of submition. This means that if you try to track members within 1 hour and 30 minutes of them logging on, Battlemetrics might not update quick enough to display the data. This will result in skewed results by showing you some but not all members.

Usage

  • If all inputs are left blank, the tracker will get the last 24 hours of Cav members that logged onto the designated server.
  • The start time references the start of an operation or when you would like to start tracking Cav members who logged on.
  • If the start time has been supplied, but the end time is left blank. The tracker will find all members from the time specified until now for today.
  • If the start time and end time are supplied, but the start and end date are left blank. The tracker will find all members who logged on from the start time until the end time for today.
  • If the start time, end time, and start date are supplied, but the end date is not. The tracker will find all members who logged on from the start time until the end time for the date specified.
  • If the start time, end time, start date, and end date are all supplied. The tracker will find all members who logged on between the start time and end time between the 2 dates specified.

Problems

  • If a start time, end time, end date are specified, but a start date is not, the tracker will error and wont return anything.
  • If the start time specified is too close to now, the tracker will return partial data and not all data.
